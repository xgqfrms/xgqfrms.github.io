## write file (single-line) 

```sh
# overwrite the file (overwrite)
$ echo "overwrite the file" > txt.txt

# just add after the last line (append)
$ echo just add after the last line" >> txt.txt

``` 

## code block (multi-lines) 

```sh
# multi-lines/block
$   (
      echo Line1
      echo Line2
      echo ...
      echo Last Line
    ) > filename.txt 
```
